# HIPO Project Page

Standalone static project page for:

**HIPO: Instruction Hierarchy via Constrained Reinforcement Learning**

arXiv: https://arxiv.org/abs/2603.16152

## Local Preview

Open `index.html` directly in a browser, or serve the directory locally:

```bash
python3 -m http.server 4173
```

Then visit:

```text
http://127.0.0.1:4173/
```

## GitHub Pages

This directory is ready to be pushed as a standalone GitHub repository. The included workflow publishes the repository root to GitHub Pages.
